1 - Crie um arquivo HTML chamado home.html
Crie dois arquivos css chamados reset.csss home.css

A ideia é criar um formulario de login utilizando flexbox
e animações do css

OBS: Não se limite a criar apenas os exemplos que vimos em aula
utilize os exemplos que vimos em aulas e a partir deles crie sua
propria tela de login com outros estilos, animações

2 - Crie uma aplicação angular utilizando angular routing 
instale o angular material

Crie os seguintes componentes:
    - Home - ng g c components/views/home
    - Login - ng g c components/views/login
    - toolbar - ng g c components/shared/toolbar

Na tela HOME fique a vontade para implementar da sua forma IDEIAS:
    listagem de pokemons
    listagem de produtos
    listagem de animais
    ...

Na tela de login utilize o mesmo estilo que utilizou no EX01 dessa lista
POREM nos inputs e botão utilize os componentes do angular material

O componente de toolbar será nossa antiga navbar que deve ser chamada no app.component.html
