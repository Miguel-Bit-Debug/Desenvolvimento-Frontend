1 - Você decidiu empreender e construir um ecommerce
e vai construir uma plataforma onde os usuários possam cadastrar produtos
e visualizar os produtos cadastrados.

Sabendo disso:

Crie uma web api (EXPRESS) que seja capaz de listar e cadastrar produtos

O produto deve conter as seguintes propriedades:
  nome: string
  descricao: string
  imagem: string
  valor: number
  categoria: string

crie os seguintes endpoints:
    listar-produtos
    adicionar-produto

os produtos devem ser salvos no mongoDB na collection de produto

OBS: Lembre-se de habilitar o CORS
